Windows 10

Source: Mollie Ruiz-Hopper, 2017, Blogs.windows.com, (https://blogs.windows.com/windowsexperience/2017/06/14/six-things-need-know-windows-10-s/)

Windows 10 , also called Threshold is one of the most used system still. It was out for retail July 29, 2015. Windows 10 included many new features that we know today. It has multiple virtual desktops, new start menu, Notification Center, and Cortana, a personal assistant that users can ask questions about the computer to. Programs like calculator, photos, and alarms were updated to be more desktop friendly. The project that was rumor, “Project: Spartan” showcased Microsoft Edge, a new browser that rivals Google and replaces Internet Explorer. Security is also upped, as apps can not set themselves as ones' default browser or app. Users are now not able to choose which updates to install, but all the updates will have to be installed. This insures that the user gets everything, so that the computer can work as intended. The Windows 10 system is pretty similar to the 7 version in terms of hardware required. 1 GHz processor, 1 gigabyte of RAM, 16 gigabytes of storage, and a graphics card of DirectX 9 or later. Windows 10 is completley free to download if upgrading from a Windows 7 or 8 but you Windows 10 normally comes with the computer a user buy now. Around 24 hours after Windows 10 was released, it already had over 14 million downloads and by March 2020, Windows 10 is running on over a billion devices. As of October 2022, the latest version of Windows 10 was released on September 30, 2022. It is succeeded by Windows 11.
