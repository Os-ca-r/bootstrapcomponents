Windows Vista

Source: Wikipedia.com, (https://en.wikipedia.org/wiki/Windows_Vista)

Windows Vista, also known as Longhorn, was released for retail October 22, 2009. Some new features include multimedia tools, a new search bar, and Windows Aero. Windows Aero is a interface that is cleaner and more appealing to the customers and users. Windows Vista was meant to be a minor update or a bridge between Windows XP and Blackcomb, but as it got more of the new features and technology, the old Longhorn was scrapped and the new Longhorn began using the Windows Server 2003 Service Pack 1 codebase. Microsoft also did a beta test on Windows Vista with hundreds of thousands of testers. It was finished being manufactured on November 8, 2006. At this point, windows were able to be downloaded and installed digitally. To install the basic version of Vista, the computer would need 512MB of memory, 15GB of space, a 800 MHz processor, and DirectX 9.0 graphics card. Windows Vista was made with the intention of having better security. It came with UAC (User Account Control), so that the user have fewer previleges by default so that malicious malware cannot make changes to the system. Windows Vista recieved not that great of reviews since it required better hardware and more restrictive licensing terms. However, they did recieve great reviews for PC gamers. Windows Vista had over 20 million copies downloaded in the first month of its release. Its final update was on October 18, 2016.
